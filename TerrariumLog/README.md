# Terrarium Log

Application SwiftUI iOS pour suivre l'évolution de vos animaux d'élevage.

## Build avec GitHub Actions

1. Poussez ce dépôt sur GitHub.
2. Ouvrez l’onglet Actions.
3. Lancez le workflow Build iOS.
4. Le build sera exécuté sur un runner macOS et vous pourrez consulter les logs.

> Pour publier sur TestFlight, il faudra ensuite ajouter une configuration de code signing et un compte Apple Developer.
